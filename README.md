Claude-coded a project to help my sister learn multiplication table, consists of a local PVP mode where you score points based on if you were correct, and an expanding table that starts as 5x5, and that you can fill to further expand it up to 10x10. 
The game saves the data inbetween the sessions in local storage of the browser.
The game is playable at https://fuczek.github.io/math-table
